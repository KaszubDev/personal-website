## Website for a cottage rental business

This project is a complete digital solution for "Domki na Mosornym", enabling their business to manage automated online reservations. It was built from the ground up, prioritizing custom functionality and a user-friendly interface based on provided PSD designs.

### 🎯 Project Goals
The main objective was to deliver a fully functional website based on professional .psd graphic designs. Alongside a pixel-perfect frontend implementation, the core requirement was to develop a proprietary booking system to handle online reservations, eliminating the need for paid third-party plugins.

### 🛠️ Tech Stack
* **Frontend:** HTML5, JavaScript & jQuery, SCSS (BEM methodology), Bootstrap
* **Backend:** Custom WordPress theme (PHP), Custom WordPress booking plugin (PHP, MySQL)
* **Integrations:** Contact Form 7 with custom styling, Google Maps

### 🌟 Key Features & Technical Highlights

#### 1. Custom WordPress Theme
I developed a completely unique WordPress theme from scratch based on PSD designs, ensuring that the final product is lightweight, fast and perfectly aligned with the brand's visual identity.

#### 2. Custom Booking Plugin
The standout feature of this project is a custom-built WordPress plugin for managing cottage reservations. Created from scratch, it allows users to check availability, select dates and finalize reservations, while giving the administration full control over bookings in the WordPress dashboard.

#### 3. Pixel-perfect PSD to HTML
The project required a precise translation of .psd designs into a fully responsive and functional website, ensuring consistent user experience across desktop and mobile devices.

#### 4. Full editorial control (ACF Pro)
I implemented a flexible content management system using ACF Pro. This allows even non-technical staff to easily update text, images and layout elements through the WordPress admin panel without any risk of breaking the site's structure.

### 💡 Challenges & Solutions

* **Custom booking logic:** The main challenge was to build this online reservation system entirely from scratch within the WordPress environment. I decided to develop a small, dedicated WordPress plugin specifically for this project. By doing so, the client avoided the need to integrate with external booking systems, saving them from potential ongoing costs and related maintenance issues.
